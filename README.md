# 3D Love Sphere - 52种语言的爱

一个美丽的3D互动网页，展示了52种不同语言的"我爱你"表达方式，排列成一个旋转的球体。

## 🌟 特性

- **52种语言**: 包含来自世界各地的"我爱你"表达方式
- **3D球体**: 使用CSS 3D变换创建真实的球体效果
- **自动旋转**: 球体持续旋转展示所有语言
- **交互效果**: 
  - 鼠标悬停暂停旋转
  - 点击文字放大查看
  - 鼠标拖拽旋转球体
- **美观设计**: 
  - 渐变背景
  - 彩虹色文字卡片
  - 浮动粒子效果
- **响应式**: 适配桌面、平板和手机屏幕

## 🚀 使用方法

1. 直接在浏览器中打开 `index.html` 文件
2. 无需安装任何依赖或运行服务器
3. 享受美丽的3D爱心球体！
